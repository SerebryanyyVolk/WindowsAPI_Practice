# 记事本说明文档

## 1. 功能说明

本项目目前实现了记事本的大部分功能，包括

1. 可用键盘操作的菜单栏
2. 文件操作

    - 新建，打开，保存，另存为
    - 可设置**打开时编码格式**(在菜单栏中点击 "文件 - 打开时编码格式" 即可设置)
    - 可选择**使用密码打开或另存为**("文件 - 使用密码打开" 以及 "文件 - 另存为加密形式")

3. 编辑操作

    - 在菜单栏中对文本进行剪切、复制、粘贴、全选、撤销，以及在文本中打印时间("编辑")
    - 可以设置文本框是否**自动换行**("查看 - 自动换行")
    - 底部有一个状态栏，会显示一些提示信息，可点击 "查看 - 状态栏" 显示或隐藏

4. 查看操作

    - 可设置窗口是否置顶，置顶后窗口会悬停在所有未置顶窗口上方("查看 - 窗口置顶")
    - 可选择三种不同的字体("查看 - 字体", 该选项仅能作用于显示体验，并不能写进文本文档中)
    - 可设置字体颜色("查看 - 颜色")

5. 简单的关于窗口

## 2. 亮点介绍

1. **新建**功能

    本项目通过ShellExecute函数启动自身实现了新建功能

2. 本项目相比于例程，将**使用密码打开和保存**、**打开时编码格式**放到了**菜单栏中文件选项的下拉栏**中，而不是每次打开或保存文件时询问。这可以提升用户在不使用这两项功能时的体验

3. 本项目专门考虑了打开属性为**只读**的文件的情况，若打开了只读文件，保存时会弹窗提示并强制只能另存为而不能直接保存。具体实现如下

    ```cpp
    //新增了如下函数，通过调用该函数可获得文件的只读信息

    //BReadLinesEx.h
    class CBReadLinesEx
    {
    //...
    public:
        bool IsReadOnly();      /* 获得文件的只读属性：返回 true 为只读
                                如果未成功打开文件，该函数默认返回 false */
    //...
    }

    //////////////////////////////////////

    //BReadLinesEx.cpp
    bool CBReadLinesEx::IsReadOnly(){
        if (hFile() != INVALID_HANDLE_VALUE)
        {
            DWORD dwAttr = GetFileAttributes(m_szFileName);
            return (dwAttr & FILE_ATTRIBUTE_READONLY);
        }
        //如果未成功打开文件，默认返回 false
        return false;
    }
    ```

4. **可以选择是否自动换行**

    实现：初始创建两个主文本框，一个可以自动换行一个不能；隐藏其中一个；用户点击 "查看 - 自动换行" 时，将显示的文本框隐藏并将隐藏的文本框显示出来，并将之前正在显示的文本框中的内容复制到当前显示的文本框中

5. **切换自动换行与保存状态检测的兼容**

    保存状态检测实现了 "当用户输入文本没有保存就直接关闭" 时，程序会弹窗提示用户尚未保存文件，并询问用户是否保存。原理为检测用户在显示文本框中改变字符就会调用函数将 isSaved 标志位置 false, 保存成功则将其置 true。

    但是改变自动换行状态也会改变显示文本框中的字符。即使用户在改变自动换行状态前已经保存，切换自动换行状态后直接关闭程序，其也会告知用户尚未保存。故本项目专门修复了该漏洞，目前切换自动换行与保存状态检测之间已经没有耦合关系。
